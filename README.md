# Minicart test

This Solution was builded using HTML5 | CSS (SCSS), ReactJS and Shopify - Themekit

## Specs

* User can add a product from the PDP page.
* Once the minicart is open the user can:
  * Remove the product.
  * Update the amount of added products.
  * Close the minicart.
